# Bike Sales Dashboard – Excel

## Overview
End-to-end Excel project demonstrating data cleaning, pivot table analysis, and dashboard visualization.

## Workbook Structure
- **Raw Data**: Original dataset before cleaning.
- **Working Sheet**: Cleaned dataset prepared for analysis.
- **Pivot Table**: Summarized KPIs (income per purchase, age brackets, commute distances).
- **Dashboard**: Interactive charts with slicers for marital status, region, and education.

## Features
- Dynamic filtering with slicers
- Income vs purchase analysis by gender
- Customer age bracket trends
- Commute distance impact on bike purchases
